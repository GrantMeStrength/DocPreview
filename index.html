🔒 Local

    Doc Preview

    Preview GitHub markdown files styled like Microsoft Learn.

    Paste a GitHub URL above and click Preview, or use Browse local to open a folder from your computer.

    Supported GitHub URL formats:

    • https://github.com/org/repo/blob/branch/path/to/file.md
    • https://raw.githubusercontent.com/org/repo/branch/path/to/file.md

    🔒 Local files never leave your device — all rendering happens in your browser, nothing is uploaded.

    GitHub preview URLs are shareable — anyone with the link will see the same document.

    🖼️ Images are not rendered. Inline images (local or GitHub-relative) will display as a labelled placeholder instead.
